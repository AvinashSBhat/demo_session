  2. Unordered list
  3. Data definition list
  4. Nested list

Ordered list

It is used to display a list of items in a specific order

ol tag is used to create an ordered list

The default order is number (1,2,3...)

type attribute helps to change the default order(type="A",'a',"i","I","1" etc.)

reversed attribute helps to reverse the order

start attribute helps to start the list from a specific number

Unordered list

It is used to display a list without any order

ul tag is used to create an unordered list

type attribute helps to change the default order(type="circle","square","disc")

Data definition list/Description list

It is used to display a list of terms and their descriptions

dl tag is used to create a data definition list

dt tag is used to define the term

dd tag is used to define a description
