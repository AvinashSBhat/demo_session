CSS(Cascading style sheets)
    :used to design/style the web pages
    :css files will save with .css extension
    :css files will be linked to the html files using the <link> tag in html file

applying css to the web pages:
    1. inline css
    2. internal css
    3. external css

1. inline css: it is used to apply the css to the web pages using the style attribute in the html file.
     ex: <p style="color:red;">Lorem ipsum dolor sit amet.</p>

Note: not recommended for large web pages
    : it is recommended for small web pages when we are applying style for a single element

selector: it is used to select the elements in the web pages

syntax:

    selector{
      property:value
      property2:value2
    }

2. internal css: it is used to apply the css to the web pages using the <style> in html file.
      ex: <style>
                p{
                  color:red;
                }
          </style>

Note: it is recommended for small/single web pages

3. external css: it is used to apply css to the web pages using the <link> tag in html file. 
               : need to create a css file with .css extension
               : css file need to link to the html file using the <link> tag in the html file
      ex: <link rel="stylesheet" href="external.css">
            rel- relationship between the css file and the html file
            href - url of the css file(hyper text reference)

Note: recommended for large web pages and multiple web pages

Box Model: used to design layout of the web pages
         : box model consist of content, padding, border and margin

content: the content of the box model
padding: the space between content and the border
border: the border of the box model
margin: the space between the box model(element) and the other box models(element)

block level elements:
              :elements occupy the full width of the web page
              :it support height and width
              :it support margin and padding

inline level elements:
              :elements do not occupy the full width of the web page
              :it does not support height and width


conversion of block to inline level elements:
          display: inline-block;- it supports height and width
          or 
          display: inline;- it won't support height and width
